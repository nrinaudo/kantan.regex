package java8

Declares kantan.regex.GroupDecoder instances for java8 date and time types.

Note that the type for default codecs might come as a surprise: the wrapping Exported is used to lower their priority. This is necessary because the standard use case will be to import kantan.regex.java8._, which brings both the instance creation and default instances in scope. Without this type trickery, custom instances and default ones would always clash.
